Установка и настройки git

Работа с файлом .gitconfig

  • git config --global user.name "Sergey Babanov"
    • ИЛИ
  • git config --global user.name "sergeybabann"
  • git config --global user.email "sergeybabann@yandex.ru"
  • git config --list

Работа с директорией

  • ls - посмотреть все файлы и папки в директории
  • ls -A - посмотреть все файлы папки в директории, а также скрытые

Работа с коммитами

  • git log - посмотреть список коммитов
  • git log --oneline - посмотреть список коммитов
  • git commit -a - эта команда включает в индекс всех измененных отслеживаемых файлов (но не новых)
  • git commit -am 'текст комментария' - это тоже самое что и -a , но можно добавить комментарий

Работа с состоянием

  • git status
  • git status -s
  • git diff - просмотр состояние изменения рабочей папки

Флаг --global делает настройки глобально, для всех пользователей
