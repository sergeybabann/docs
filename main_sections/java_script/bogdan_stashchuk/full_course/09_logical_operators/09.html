9 Логические операторы

Оператор НЕ

Оператор !(не) или оператор !(отрицания)

Оператор ! чаще всего используется в условных инструкциях.

Посмотрим примеры с оператором НЕ: 

                    
                        !10 // false 
                        !0 // true 
                        !'abc' // false
                        !'' // true
                        !true // false
                        !undefined // true
  • !10 // false (не 10)
  • !0 // true (не 0, ноль это ложное значение)
  • !'abc' // false (не пустая строка, пустая строка это ложное значение)
  • !'' // true (не пустая строка, пустая строка это ложное значение)
  • !true // true (не true)
  • !undefined // true (не undefined)

Оператор !(не) всегда дает 1 из 2-ух значений. Либо true, либо false.

Оператор (не) - это префиксный оператор, и это унарный оператор.
