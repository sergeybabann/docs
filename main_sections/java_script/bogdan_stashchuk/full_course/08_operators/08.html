Операторы

Оператор - это встроенная в JavaScript функция. То есть внутренняя функция JavaScript. И когда вы используете тот илиной оператор, вы по сути запускаете ту или иную встроенную функцию. И эта функция выполняет определенные действия, и так же возвращает результат, как любая функция.

Виды операторов

  • Арифметические операторы:
    + плюс
    - минус
    * умножение
    / деление
  • Операторы сравнения:
    === равно
    !== не равно
    <= меньше либо равно
    >= больше либо равно
  • Логические операторы:
    ! не
    && и
    || или
  • Операторы присваивания:
    = присваивание

Кроме этих операторов существует много других операторов. Но это самые основные.

  • Символьные операторы:
    typeof
    instanceof
    new
    delete

Рассмотрим примеры

Оператор запятая. С помощью этого оператора вы можете объявить несколько переменных в одной строке. 

                    
                        let a, b // оператор запятая

Оператор присвавания. С помощью этого оператора можно присвоить той или иной переменной то или иное значение.

В этом примере переменной a присваивается число 10, а переменной b присваивается значение переменной a, и эта переменная представляет собой выражение. 

                    
                        a = 10 // оператор присваивания
                        b = a // оператор присваивания

В этом примере сначала мы получим результат этого выражения a + b, и только после срабатывает оператор присваивания let c =
И результат присвоится переменной c, которая будет объявлена. 

                    
                        let c = a + b // 2 оператора, оператор присваивания и оператор плюс

И после всего этого при вызове в консоль console.log(c) , мы получим 20. Это сумма переменных 

                    
                        console.log(c) // 20
Операторы присваивания
                    
                        a = 10 // оператор присваивания

У оператора присваивания есть 2 операнда. Слева и справа от оператора присваивания - это операнды. Оператор - это встроенная функция. И эта встроенная функция выполняет какие-то действия и возвращает значение. Можно ее изобразить схематично:

function =(переменная, выражение) {

  1. 1) получение результата выражения
  2. 2) поиск переменной по имени
  3. 3) присваивание результата выражения переменной
  4. 4) возврат результата выражения
}

= это название функции, и эта функция внутренняя. У функции есть 2 параметра: 1-ый параметр - это переменная, которую вы передаете оператору присваивания как левый операнд, 2-ой параметр - это выражение, он находится справа. Получив значение для этих 2-ух параметров - название переменной и само выражение, которое необходимо сначала интерпретировать, внутри функции JavaScript выполняет ряд действий.

1) То есть сначала JavaScript получает результат выражения, которое получено как 2-ой параметр здесь, и это цифра 10,
2) далее JavaScript пытается найти переменную по имени, (имя переменной) a
3) результат выражения, который был получен на шаге 1) мы присваиваем переменной, которая была найдена на шаге 2)
4) в итоге эта внутренняя функция возвращает результат выражения, полученного на 1) этапе

И в данном примере эта функция вернет значение 10

Унарные и бинарные операторы

Унарные операторы

У унарных операторов всегда 1 операнд(он же аргумент). Такой оператор как ++ является унарным оператором. Потому что у него только 1 операнд, и такой оператор увеличивает значение на единицу. 

                    
                        a++ // унарный оператор с операндом a

Оператор +
Это тоже унарный оператор, и с помощью него можно конвертировать строку в число. 

                    
                        +a // оператор плюс

Оператор delete, это тоже унарный оператор. И у него один операнд. И это по сути путь к свойству того или иного объекта. 

                    
                      delete obj.a

Оператор typeof, это унарный оператор. Здесь a это операнд. С помощью этого оператора можно узнать тип той или иной переменной. 

                    
                      typeof a

Оператор new, с помощью этого оператора можно создать новый экземпляр того или иного объекта. 

                    
                        new Object()

Бинарные операторы

У бинарных операторов всегда 2 операнда(аргумента). 

                    
                        a = 5 // оператор присваивания
                    
                
                    
                        a + b // оператор плюс, сложение

Оператор увеличения значения. Мы берем значение переменной a, увеличиваем на 5, и присваиваем результирующее значение переменной a. То есть мы меняем значение и присваиваем его как новое значение переменной. 

                        
                            a += 5 // оператор увеличения значения

Оператор сравнения. 

                    
                        a === b // оператор сравнения

Мы сравниваем значение 2-ух переменных. Этот оператор сравнивает как тип, так и значение. То есть если вы попытаетесь сравнить строку 5 и число 5, то результатом выполнения оператора сравнения будет false. Потому что строка 5 не равна цифре 5. Если вы срвните 2 строки, вот так: abc === abc, то получите true, потому что эти строки одинаковы.

Оператор и, логический оператор. 

                    
                        a && b // оператор и, логический оператор

Это также унарный оператор, у него 2 операнда.
