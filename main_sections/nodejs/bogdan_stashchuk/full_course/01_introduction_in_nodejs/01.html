1 Введение в NodeJS

Введение в NodeJS

NodeJS - это среда выполнения кода вне браузера. Это серверная технология, которая позволяет выполнять код JavaScript непосредственно на серверах, вне клиентских устройств. С помощью NodeJS создается API для различных библиотек и фреймфорков JavaScript.

NodeJS лучше всего подходит для обработки потоковых данных в реальном времени. Для выполнения задач, которые должны быть выполнены быстро, запрос-ответ. Например работа с потоками баз данных. Также для выполнения задач, которые не трубуют большого количества ресурсов процессора. Поэтому NodeJS хорошо подходит для создания API сервиса.

Где Nodejs не стоит использовать?

NodeJS не лучший выбор для обработки видео и для машинного обучения.

Отличия использования JavaScript в веб-браузере и в NodeJS

В веб-браузере есть глобальный объект window, и у этого объекта есть много различных свойств. И объект console в веб-браузере является одним из свойств глобального объекта window. В NodeJS глобальный объект global, и свойства этих 2-ух глобальных объектов отличаются, потому что в веб-браузере и на сервере выполняются различные задачи.

В веб-браузере с помощью JavaScript можно взаимодействовать с DOM. В NodeJS понятие DOM отсутствует, потому что нет веб-браузера, здесь работа на сервере. Следовательно нет взаимодействия с веб-браузером, нет взаимодействия c DOM.

В веб-браузере код нужно адаптировать под различные веб-браузеры, так как их большое количество, и не все они сразу начинают поддерживать новые функции JavaScript. А потому необходимо конвертировать код JavaScript так, чтобы его понимали большинство веб-браузеров. В NodeJS вы можете использоватьвсе функции, которые поддерживаются определенной версией NodeJS. Для выполнения кода в NodeJS не требуется какая-либо конвертация или компиляция кода JavaScript для того, чтобы NodeJS корректно выполнила этот код.

В веб-браузерах нет доступа к файловой системе клиентов. То есть веб-приложение не может, например, создавать новые файлы у клиента, либо же удалять файлы, либо менять настройки операционной системы - это запрещено. Веб-приложения работают изолировано в веб-браузере. А вот в приложениях NodeJS есть полный доступ к файловой системе. Можно взаимодейтствовать с операционной системой, сетью, менять их настройки. Это делается встроенными в NodeJS функциями.
