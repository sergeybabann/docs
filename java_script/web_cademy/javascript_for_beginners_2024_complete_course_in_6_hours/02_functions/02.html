2 Функции

Объявление функции. 2 способа.
DRY - don't repeat self (не повторяй себя)
Объявление функции и ее вызов в консоли:
  • Объявление функции (в теле функции метод console.log)
  • вызов функции: имя функции
                    
                        function sayHi() {
                            console.log('Привет, дорогой пользователь!')
                        }
                        
                        sayHi()
Существует 2 способа объявления функции:
  • function declaration
  • function expression
function declaration 
                    
                    // function declaration
                    function sayHi() {
                        console.log('Привет')
                    }
function expression 
                    
                    // function expression

                    // let
                    let sayHi = function() {
                        console.log('Привет')
                    } 

                    // const
                    const sayHi2 = function() {
                        console.log('Здравствуйте')
                    }

                    // var
                    var sayHi3 = function() {
                        console.log('Приветствую')
                    }
function declaration:
Можно вызывать функцию до того момента, как она была объявлена в коде. 
                    
                        // вызов функции
                        sayHi() 
                        
                        // объявление функции
                        function sayHi() {
                            console.log('Приветики')
                        }
function expression:
Нельзя вызывать функцию до того момента, как она была объявлена в коде. Будет ошибка. 
                    
                        // вызов функции не возможен
                        sayHi()

                        // объявление функции
                        const sayHi = function() {
                            console.log('Привет')
                        }  
                    
                
                    
                        // объявление функции
                        const sayHi = function() {
                            console.log('Привет')
                        }  

                        // вызов функции возможен
                        sayHi()
Какие аргументы для того, чтобы использовать function declaration ?
  • Это короче и лаконичней
  • Мы можем описать работу программы, а потом ниже описать те функции, которые нам нужны
Какие аргументы для того, чтобы использовать function expression ?
  • Если функция вызывается, то она объявлена где-то выше. То есть более строгие правила, они и более надежные. Не объявленная функция ранее в коде, не может быть вызвана. Это строгое правило.
Параметры и аргументы для функции
Функция sayHi принимает в скобках параметр name. Параметр name объявлен в функции. Значение параметра функции 'Юрий', которое находится в скобках передается в параметр name для функции sayHi и далее идет в переменную в шаблонной строке в методе console.log
Измените значение аргумента функции 'Юрий' и посмотрите результат в консоли. 
                    
                        function sayHi(name)// name это параметр функции {
                            console.log(`Привет, ${name}! Как твои дела?`)
                        }
                        
                        sayHi('Юрий')// Юрий это аргумент функции
Меняем значение в аргументе функции, смотрим результат. 
                    
                        function sayHi(name)// name это параметр функции {
                            console.log(`Привет, ${name}! Как твои дела?`)
                        }
                        
                        sayHi('Юрий')
                        sayHi('Александр')
                        sayHi('Павел')
return
Функция ничего не возвращает. В ней нет return
  • функция summ принимает параметры: a, b
  • константа result присваивает результат сложения этих двух параметров
  • вывод в консоль значения константы result
  • summ передает аргументы 10 и 15 в функцию summ (10 идет на параметр a, 15 идет на параметр b, то есть по порядку)
Функция отработала результат сложения параметров, но ничего не вернула. Поэтому она undefined 
                    
                        function summ(a, b) {
                            const result = a + b;
                            console.log(result)
                        }
                        
                        summ(10, 15) // 10 идет на a, 15 идет на 15,то есть по порядку
Функция возвращает результат. В ней есть return
  • функция summ принимает в свои параметры a, b значения аргументов 10, 15
  • константа result присваивает результат сложения этих двух параметров
  • константа result возвращает результат сложения
  • переменная let присваивает результат сложения аргументов 10, 15, которые идут на параметры функции
  • метод console.log выводит результат сложения, который сохраняется в переменной res
Остановка действий функции происходит после того, как она совершила возврат (return). Поэтому писать код после return бессмысленно. Код находящийся ниже return не работает. 
                    
                        function summ(a, b) {
                            const result = a + b;
                            return result;
                        }
                        
                        let res = summ(10, 15);
                        console.log(res);
Функция как аргумент
                    
            // функция как аргумент
            function summ(a, b) {
                return a + b
            }
                        
            console.log(summ(15, 15))
            // выводится функция с результатом сложения аргументов 15, 15
                    
                
                    
                        // функция как аргумент
                        function summ(a,b) {
                            return a + b
                        }
                        
                        const result = summ(summ(15, 15), summ(20, 20))
                        console.log(result)                       
                    
                
                    
                        // функция как аргумент
                        function summ(a,b) {
                            return a + b
                        }
                        
                        function diff(a, b) {
                            return a - b
                        }
                        
                        function doSomething(func) {
                            let x = 10
                            let y = 15
                            let result = func(x, y) 
                            console.log(result)
                        }
                        
                        doSomething(summ) // 25
                        doSomething(diff) // -5
