1 Основы

Переменные
Самый простой способ узнать значение переменной это вывести ее в консоль:
Объявление переменной userName, присвоение ей значения 'Марк'. Вывод переменной в консоль. 
                    
                        let userName = 'Марк';
                        console.log('userName');

Объявление и переопределение значения переменной:

Объявление переменной userName и присвоение ей значения Марк.
Затем присвоение ей других значений:
  • Егор
  • Анжела
                    
                    let userName = 'Марк';
                    console.log(userName);
                    userName = 'Егор';
                    console.log(userName);
                    userName = 'Анжела';
                    console.log(userName);
Типы переменных
Существует 3 типа переменных: let, const, var

Переменная var уже используется редко, только на старых проектах, где остался код со старых времен. В старой версии es5 языка javascript применялся синтаксис перемнной var.
Переменная let это новый синтаксис, который применяется в следующей версии языка javascript es6.
Переменная const это константа, ей нельзя назначить новое значение внутри программы, так как она по умолчанию неизменна.

переменная let 
                    
                    let userName = 'Марк';
                    console.log('userName');
переменная var 
                    
                    var userName = 'Петр';
                    console.log('userName');
переменная const 
                    
                        const yearOfBirth = 1990;
                        console.log('yearOfBirth');
Попытка присвоения нового значения перемнной типа const.
Консоль выдаст ошибку с текстом:
Uncaught TypeError: Assignment to constant variable 
                    
                        const yearOfBirth = 1990;
                        yearOfBirth = 2000;
Переменная типа let может объявляться без присвоения значений, эти значения могут быть присвоены ей позже.
Объявление переменных без присвоения значений. Присвоение значений производится следующим шагом. 
                    
                        let name, age;
                        name = 'Юрий';
                        age = '29';

Переменной типа const после ее объявления сразу нужно присвоить значение. 

                    
                        const height = 180;

var - старый формат объявления переменной. В отличие от let и const имеет функциональную область видимости, а не блочную. Будет рассмотрен в отдельном уроке по нюансам работы JS не рекомендуется к использованию. Может менять значение на протяжении работы программы.

Область видимости
Область видимости внутри блока кода

Переменные типа let и const имеют область видимости внутри блока кода.

Переменная userName выводится в консоль только когда она находится внутри блока кода. блок кода - это все то, что находится внутри фигурных скобок. Переменная userName, которая вынесена за пределы блока кода не выводится в консоль. Консоль выдает ошибку с текстом Uncaught ReferenceError: userName is not defined 
                    
                        {
                            let userName = "Bob"
                            console.log(userName)
                        }
                            console.log(userName)
Типы данных
Существует несколько типов данных
Перечислим эти типы:
  • string (строка)
  • number (число)
  • boolean (логический)
  • null (ничего, не известно)
  • undefined (не определено)
  • object (объект)
  • BigInt(создание больших чисел)
  • Symbol (для работы с объектами)
string (тип данных строка)
Любой текст, любой набор символов внутри кавычек. 
                    
                        let userName = 'Mark'
                        let age = '30'
number (тип данных число)
Любое число без кавычек. 
                    
                        let age = 30
boolean (тип данных логический)
Всего 2 значения - true и false 
                    
                        let isMarried = true 
                        let devorced = false
null (тип данных не известно)
Пусто, ничего. 
                    
                        let height = null
undefined (тип данных не определено) переменная объявлена, но ее значение не определено.
                    
                        let userName2;
BigInt (тип данных для работы с большими числами)
В первой строке метода console.log консоль выводит максимально большое число, c которым может работать тип number.
Во второй строке к числу дописывается символ n, это преобразовывает число в тип данных BigInt.
Тип BigInt не может работать в связке с типом number, поэтому все операции по взаимодействию больших чисел необходимо соовершать только в рамках одного типа - BigInt 
                    
                        console.log(Number.MAX_SAFE_INTEGER)
                        console.log(90071992547409919007199254740991n);
тип Symbol - это уникальное примитивное значение. Оно позволяет добавлять уникальное свойство к объекту с увереностью, что он не перезапишет никакое другое свойство, уже существующее у объекта.
Возможности языка javascript (динамическая типизация)

Возможности языка javascript позволяют на лету переопределять один тип данных на другой без длительных операций. Переменная someVariable из типа данных string мгновенно становится типом данных number, затем boolean. Достаточно применения одного оператора =
Но не стоит этим злоупотреблять. Резко менять тип данных одной переменной без лишней необходимости не нужно, лучше создать новую переменную и присвоить ей свой тип. 

                    
                        let someVariable = 'Some value';
                        someVariable = 50;
                        someVariable = true;
1-ая строка тип данных string. Объявление переменной и присвоение ей значения.
2-ая строка тип данных number. Присвоение переменной нового значения типа данных number.
3-я строка тип данных boolean. Присвоение переменной нового значения типа данных boolean.

А также наименование переменных должно быть понятным. Не стоит давать имена переменным которые не отображают смысл и сущность самой переменной. Так как потом можно забыть к чему эта переменная относится.

Оператор typeof

Оператор typeof позволяет определить тип переменной

Две переменные выводятся в консоль. Оператор typeoff в методе condole.log показывает тип данных в консоли.
Переменная someName - number
Переменная someName2 - string 
                    
                        const someName = 44;
                        const someName2 = 'Bob';
                        console.log(typeof someName);
                        console.log(typeof someName2);
Операторы сравнения и условия
Подзаголовок
Два примера сравнения чисел. Операторы сравнения производят сравнение чисел. 10 больше 5? Да 10 больше 5. Javascript сравнил эти два числа, и вернул результат сравнения. Результат сравнения true(истина).
Во втором сравнении результат false(ложь), так как 20 не больше 80. 
                    
                        console.log(10 > 5) // true
                        console.log(20 > 80) // false
Объявлена переменная let result2, ей присвоено значение false. Оно присвоено в результате вычисления операции сранения двух чисел.
Операция сравнения - 10 равно 5. Результат сранения false (ложь). Почему ложь? Потому, что 10 не равно 5. Результат возвращается в переменную, записывается в нее, и в методе console.log выводится в консоль.
Таким образом переменная result2 имеет булевое значение false 
                    
                        let result2 = 10 == 5 // 10 равно 5 ? нет 
                        console.log(result2) // false
Динамическое приведение типов
По-скольку Javascript не строго типизированный язык, то в нем сущеествует динамическое приведение типов. Например, если мы сравним 5-ку в виде строки с 5-кой в виде числа, при этом применим оператор сравнения == (не строгое сравнение), то произойдет приведение типов и результат будет true 
                    
                        console.log('5' == 5) // true
Второй такой же пример, но здесь мы применяем оператор сравнения === (строгое сравнение). Результат false
Этот оператор сравнения не приведет разные типы данных к приведению. То есть 5 число не равно 5 строке. Разные типы данных. Результат сравнения false
Строгое сравнение === 
                    
                        console.log('5' === 5) // true
Операция присвоения. В данном случае консоль выведет следующий результат:
Uncaught SyntaxError: Invalid left-hand side in assignmen
Консоль выводит ошибку потому, что операция присвоения написана некорректно. 
                    
                        console.log('5' = 5) // true
!!!Внимание Нельзя путать операторы == и оператор =
Это разные операторы.
== оператор сравнения
= оператор присваивания
Оператор присвоения
Оператор присвоения ( = ) выполняет присвоение значения. Например переменной number было присвено значение 5, а затем присвоено значение 10. Этот оператор выполнил операцию присвоения нового значения типа данных number. Было 5 стало 10. 
                    
                        let number = 5
                        number = 10
                        console.log(number) // 10

                        const time = 10;
