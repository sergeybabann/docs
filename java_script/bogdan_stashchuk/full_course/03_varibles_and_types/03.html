3 Переменные и типы

Переменные

Переменные дают возможность повторного доступа к значениям, которые присвоены этим переменным.

Именование переменных. Существует 3 подхода к именованию переменных в JavaScript
  • PascalCase - используется для того, чтобы называть типы и классы в JavaScript. Например есть такой класс как объект, либо array(массив), они начинаются с заглавной буквы. Потому, что с помощью классов создаются экземпляры классов.
  • DB_PASSWORD - Такой подход к названию переменных используется тогда, когда значения, которые мы присваиваем переменным известны до запуска приложения и не меняются на протяжении работы приложения. Такие значения можно сказать константы. Например такой переменной можно присвоить значение пароля или логина для доступа в базу данных.
    DB_PASSWORD
    DB_USER и т.д.
  • camelCase - такое именование переменных рекомендуется использовать во всех остальных случаях, и используется чаще всего.

Названия переменных должны быть понятными. В самом названии должна быть заключена информация относящаяся к сущности перемнной. Например: myName = "Сергей", myEmail = "sergeybabann@yandex.ru"
Лучше использовать название более длинным и понятным.

Объявление переменных

В JavaScript есть 3 ключевых слова с помощью которых объявляются переменные:

let
const
var 

                    
                        let a // Объявление переменной с названием a
                    
                
                    
                        const с = 10 // Объявление переменой с и присваивание ей значения 10
                    
                
                    
                        a = true // Присваивание значения переменной

Переменной типа let можно присваивать новые значения, чего нельзя делать для переменной типа const
Переменной типа const должно быть присвоено значение сразу же при ее объявлении: 

                    
                        const c = 10
